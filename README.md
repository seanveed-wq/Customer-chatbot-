# AI SaaS Chat UI

ChatGPT-inspired but original AI SaaS chat interface.

## Features
- Dark mode first
- Responsive (mobile + desktop)
- Sidebar + chat layout
- Lightweight (HTML, CSS, JS)
- Ready for AI / API integration

## How to Run
1. Download or clone repo
2. Open `index.html` in browser

## Use Case
- AI chat apps
- Customer support bots
- SaaS dashboards
